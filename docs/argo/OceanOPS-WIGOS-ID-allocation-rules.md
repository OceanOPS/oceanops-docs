# OceanGliders
## first deployment
To get a WMO, users must request it through the API. The API create a record in our data base under the status "PROBABLE". This will be the first mission of the glider. At the same time, it deliver the WIGOS ID.
So first deployment when declared after using the "request id tool" have a WIGOS ID allocated to it.

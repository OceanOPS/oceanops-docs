# Upload platform
To upload a new platform of FVON, we receive a request from a FVON program manager (or data manager of the program).
This request should come with the minimum set of metadata we need: 
* FVON program, Ship name, ship country, vessel gear type (sensor type), sensor model, start date, telecom type. 
* IMO number, mmsi and call sign are also very usefull.

Once we got this set of information, the ship must be created by the TC (or program manager). 
In the setting section "Hide metadata" option is the standard, and "validated" should be selected if checking on Marine Traffic and Vessel finder has been done.

Then the platform can be registered through csv upload (the only one option tested so far). All fishing vessel platform must be tagged FVON.

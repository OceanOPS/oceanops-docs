# FVON - GEAR TYPE, PLATFORM, SENSOR
## GEAR TYPE
For FVON, GEAR TYPE correspond to our PLATFORM TYPE. They are define here : https://openknowledge.fao.org/server/api/core/bitstreams/830259c5-cbba-49f8-ae0d-819cd54356d3/content.

As for SOT, the ship is not the platform. Metadata about ship does not refer to the gear type, but higer level (fishing vessels, Identifiers, etc.)

## SENSOR MODELS
Sensor models refers to the manufacturers and the type of sensors used by FVON. They can be cross programs.
